# Tetris 🎮

Tetris feito do zero com HTML, CSS e JavaScript puro — sem bibliotecas.

## Como jogar
- ← → Mover
- ↑ Girar
- ↓ Descer
- Espaço Hard drop
- P Pausar

## O que foi usado 
- Canvas API (ctx.fillRect, getContext)
- Game loop com requestAnimationFrame
- Detecção de colisão com matriz 2D
- Rotação de matriz (transposição)
- Manipulação do DOM com classList
